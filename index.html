📋 Project Summary

Overview

FifthProj is a Java application that demonstrates modern Java features including Records (Java 14+), Streams API, and functional programming. The project focuses on performing various database operations and data analysis on user registration records stored in a MySQL database.

Core Components

  • MnClsA - Main class with 8 query methods demonstrating Streams API usage
  • DbUtilsCls - Database utility class with 3 data retrieval methods
  • Register - Java Record for user registration data
  • MyView - Java Record for view data with custom formatting
  • IFaceDbUtil - Interface defining database configuration and SQL queries
  • TestCls - Test class for verifying Record functionality

Key Features

  • Stream-based filtering and sorting of records
  • Statistical analysis (max, min, average, sum, count)
  • Functional programming with Lambda expressions
  • Prepared statements and callable statements for database operations
  • Immutable data structures using Java Records
  • Support for both direct queries and stored procedures

Technologies Used

  • Java 14+ (Records, Stream API)
  • MySQL Database
  • JDBC for database connectivity
  • Functional Programming Paradigm
