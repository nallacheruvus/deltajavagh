📦 Package: com.sa.cls

This package contains the main classes for performing query operations and testing functionality.

CLASS
MnClsA

Main class for performing various query operations on Register records.

This class demonstrates the use of Java Streams API and functional programming to perform filtering, sorting, and statistical operations on user registration data. It connects to a database using DbUtilsCls and processes Register records.

Key Functionality:

  • Statistical analysis (max, min, average, sum, count) of record IDs
  • Filtering records by specific criteria (ID values, name patterns)
  • Sorting records in ascending or descending order
  • Displaying view data from the database

Methods:

public static void main(String[] args)
Performs statistical analysis on Register records including maximum, minimum, average ID value, total count, and sum of all ID values.
public static void firstQuery(Stream<Register> strRegs)
Initial query entry point that displays all register records.
public static void secQuery(Stream<Register> strRegs)
Sorts registers in descending order by ID.
public static void thirdQuery(Stream<Register> strRegs)
Filters and displays registers with odd ID values (1, 3, 5, 7, 9).
public static void fourthQuery(Stream<Register> strRegs)
Filters and displays registers with even ID values (2, 4, 6, 8, 10).
public static void fifthQuery(Stream<Register> strRegs)
Filters and displays registers whose names start with 'J'.
public static void sixthQuery()
Retrieves and displays MyView records from the database.
public static void sevenQuery(Stream<Register> strRegsa)
Displays all registers from the provided stream.

📦 Package: com.sa.db

Database utility package containing operations for database access and data retrieval.

CLASS
DbUtilsCls

Database utility class for performing database operations.

This class implements IFaceDbUtil and provides methods to:

  • Execute queries to retrieve Register records from the database
  • Fetch MyView records from a database view
  • Execute stored procedures for advanced data retrieval

It maintains a static database connection using MySQL JDBC driver and provides methods to map ResultSet data to Java objects.

Database Connection Details:

Property Value
Driver MySQL Connector/J (com.mysql.cj.jdbc.Driver)
Database deltajavamarch
Connection URL jdbc:mysql://localhost:3306/deltajavamarch
Username root

Methods:

public List<Register> retRegs()
Retrieves all Register records from the database. Executes a PreparedStatement query and maps each ResultSet row to a Register object.
Returns: List of Register objects containing all database records
public List<MyView> retList()
Retrieves MyView records from the database view. Executes a query to fetch records from the myyiew database view.
Returns: List of MyView objects containing view records
public List<Register> retRegsProc()
Retrieves Register records using a stored procedure. Executes the 'myproc' stored procedure and maps each ResultSet row to a Register object.
Returns: List of Register objects returned by the stored procedure

Author: Development Team | Version: 1.0

📦 Package: com.sa.recs

Record classes for data representation and data transfer objects.

RECORD
Register

Record class representing a user registration entry.

This Java record encapsulates user registration data:

Fields:

Field Name Type Description
id int Unique identifier for the user
name String User's full name
email String User's email address
mobile String User's mobile phone number

Notes:

The record is immutable and provides accessor methods for each field. It's primarily used for mapping database records to Java objects and for stream-based data processing operations.

Author: Development Team | Version: 1.0

RECORD
MyView

Record class representing a user view with personal information.

This Java record encapsulates user view data:

Fields:

Field Name Type Description
name String User's full name
email String User's email address
mobile String User's mobile phone number
raddress String User's residential address

Methods:

Provides a formatted string representation of the MyView record with all fields displayed in a readable multi-line format.
