📖 Project Overview

FifthProj is a Java application demonstrating modern Java features including Records, Streams API, and functional programming paradigms. The project focuses on performing various database operations and data analysis on user registration records.

Project Structure

  • com.sa.cls - Main query classes and testing functionality
  • com.sa.db - Database utilities and connection management
  • com.sa.recs - Data record classes (Register, MyView)
  • comm.sa.inte - Interface definitions and configuration

Key Technologies

  • Java Records (Java 14+)
  • Streams API for functional programming
  • MySQL Database
  • JDBC for database connectivity
  • Lambda Expressions and Method References

MnClsA

Description

Main class for performing various query operations on Register records. This class demonstrates the use of Java Streams API and functional programming to perform filtering, sorting, and statistical operations on user registration data.

🎯 Usage Guide

Running the Application

java -cp bin com.sa.cls.MnClsA

Testing Register Record

java -cp bin com.sa.cls.TestCls

Example: Creating a Register

var register = new Register(1, "John Doe", "john@example.com", "1234567890"); System.out.println(register.id()); // Output: 1 System.out.println(register.name()); // Output: John Doe System.out.println(register.email()); // Output: john@example.com System.out.println(register.mobile()); // Output: 1234567890

Example: Processing with Streams

DbUtilsCls db = new DbUtilsCls(); List<Register> registers = db.retRegs(); registers.stream() .filter(r -> r.name().startsWith("J")) .sorted((a, b) -> a.id() - b.id()) .forEach(System.out::println);
