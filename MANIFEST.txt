                            PROJECT OVERVIEW
================================================================================

Java Application demonstrating modern Java features:
  • Java Records (Java 14+)
  • Streams API for functional programming
  • Lambda expressions and method references
  • MySQL database integration via JDBC

Core Purpose:
  Perform various query operations, data filtering, sorting, and statistical
  analysis on user registration records stored in a MySQL database.
